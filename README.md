# <img src="https://upload.wikimedia.org/wikipedia/commons/a/a5/Instagram_icon.png" width="28"> Instagram Clone (Desktop UI)

A front-end clone of Instagram built using **HTML** and **CSS**, focused on replicating the visual design and layout of the platform for laptop/desktop screens.

---

## 🚀 Features

* 📌 Instagram-like user interface
* 🧭 Navigation bar layout
* 📰 Post feed design
* 🎨 Clean and modern styling
* 🏗️ Structured using semantic HTML

---

## ⚠️ Responsiveness

> This project is currently optimized for **laptop/desktop screens only**.
> 📱 Mobile responsiveness is not implemented yet.

---

## 🛠️ Tech Stack

* HTML5
* CSS3
